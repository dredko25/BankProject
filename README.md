## Windows Forms застосунок для управління банківськими рахунками. Дозволяє створювати рахунки, вносити та знімати кошти.

## Функціонал

- Створення банківських рахунків (звичайні та з відсотковою ставкою)
- Поповнення рахунку
- Зняття коштів
- Відображення всіх рахунків у таблиці

## Як використовувати

1. Введіть ім'я власника у відповідне поле
2. (Опціонально) вкажіть відсоткову ставку
3. Натисніть "Create Account" для створення рахунку
4. Оберіть рахунок у таблиці для виконання операцій
5. Введіть суму та натисніть "Deposit" або "Withdraw"

## Технології

- C# / .NET
- Windows Forms
- List<T> для зберігання даних

## Класи

- `BankAccount` - базовий клас рахунку
- `SavingAccount` - рахунок з відсотковою ставкою (наслідується від BankAccount)
